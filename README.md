Phase 1: Basic Fundamentals
เน้นการปูพื้นฐานเรื่อง UI และโครงสร้างพื้นฐานของ Flutter

Basic 1 UI and Layout: เรียนรู้วิธีการจัดหน้าจอด้วย Widget พื้นฐาน เช่น Container, SizedBox, Row, Column, Stack, Position, Icon, Color, EditTextField, Image, SingleChildScrollView, Button และการจัดการ Event ต่างๆ

Basic 2 UI and Flow: เข้าใจโครงสร้างโปรเจกต์ ตั้งแต่จุดเริ่มต้น main, การใช้ MaterialApp, และ Scaffold (รวมถึง AppBar, Body, FloatingActionButton)

Basic 3 Navigation: การจัดการการเปลี่ยนหน้า (Page Navigation)

Basic 4 Stateless vs Stateful: ความแตกต่างระหว่าง Widget ที่ไม่มีการเปลี่ยนแปลงสถานะและที่มีการเปลี่ยนแปลงสถานะ

Basic 5 Collection UI: การแสดงผลข้อมูลแบบรายการผ่าน ListView, GridView และการสร้าง Custom Cell

Basic 6 Custom Widget: การสร้าง Widget ใช้เองและการสื่อสารระหว่าง Widget

Basic 7 setState & Performance: เข้าใจการทำงานของฟังก์ชัน setState และผลกระทบต่อ Performance ของแอปฯ

Basic 8 Platform Handling: การจัดการและปรับแต่งแอปฯ ให้รองรับ iOS, Android และ Desktop

Phase 2: Networking & Data
เน้นการเชื่อมต่อแอปพลิเคชันกับโลกภายนอกผ่าน API

Network (Dio): การใช้ Library Dio เพื่อจัดการ HTTP Method ต่างๆ ทั้ง GET, POST, DELETE, PUT

Data Modeling: การแปลง JSON เป็น Model (แนะนำเครื่องมืออย่าง quicktype.io)

Data Fetching: ฝึกดึงข้อมูลจาก Web API (เช่น jsonplaceholder.typicode.com)

Error Handling: การจัดการ Exception และข้อผิดพลาดต่างๆ ที่เกิดจากการเชื่อมต่อเครือข่าย

Phase 3: State Management
การจัดการสถานะของข้อมูลในแอปพลิเคชันที่มีความซับซ้อนสูง

Phase 4: Deployment & Logging
ขั้นตอนสุดท้ายคือการนำแอปฯ ออกสู่สายตาผู้ใช้งาน

Deployment: การเตรียมตัว Build แอปฯ สำหรับ Android และ iOS

Google Play: ขั้นตอนการนำแอปฯ ขึ้น Google Play Store

App Store: ขั้นตอนการนำแอปฯ ขึ้น Apple App Store

Logger: การทำระบบบันทึก Log เพื่อใช้ตรวจสอบการทำงานและหาข้อผิดพลาดของแอปฯ
